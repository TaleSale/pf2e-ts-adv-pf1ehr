# Тест новой функциональности @Rebellion enricher

## Примеры использования с кнопкой отправки в чат

### Проверки с КС:
- @Rebellion[type:loyalty|dc:15] - Проверка верности КС 15
- @Rebellion[type:security|dc:20] - Проверка безопасности КС 20  
- @Rebellion[type:secrecy|dc:25] - Проверка секретности КС 25

### Проверки против известности:
- @Rebellion[type:notoriety] - Проверка против текущей известности
- @Rebellion[type:notoriety/2] - Проверка против половины известности
- @Rebellion[type:notoriety*2] - Проверка против удвоенной известности

### Проверки против опасности:
- @Rebellion[type:dangers] - Проверка против текущей опасности
- @Rebellion[type:notoriety+dangers] - Проверка против известности + опасность

## Как это работает:

1. **Основная кнопка** (слева) - выполняет бросок немедленно, как раньше
2. **Кнопка чата** (справа, с иконкой комментария) - отправляет в чат сообщение с кнопкой для броска

### Преимущества кнопки чата:
- Позволяет игрокам видеть, какой бросок нужно сделать
- Показывает формулу и КС перед броском
- Можно использовать для координации с другими игроками
- Сохраняет историю запросов на броски в чате

### Пример использования:
1. ГМ пишет в журнале: "Сделайте @Rebellion[type:loyalty|dc:15]"
2. Игрок нажимает на кнопку чата (справа)
3. В чат отправляется сообщение с деталями броска и кнопкой "Бросить"
4. Игрок (или ГМ) нажимает "Бросить" для выполнения броска
